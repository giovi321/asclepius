---
title: "Timeline"
---

A chronological view of all documents for the selected patient, grouped by
year on a vertical track with a mini-map sidebar and a floating date picker
for jumping to a specific period.

Cards show the best available date (visit > issue > doc date), document
type, filename, doctor/facility, a short summary, and a colored banner when
the document is linked to a medical event.

### Color coding

| Color          | Document types                                               |
| -------------- | ------------------------------------------------------------ |
| Primary (blue) | Specialist reports, discharge letters, ER reports, surgical reports |
| Blue           | Blood tests, lab tests                                       |
| Emerald        | Invoices, receipts, insurance claims                         |
| Purple         | Radiology reports, DICOM imaging                             |
| Orange         | Prescriptions                                                |
| Teal           | Vaccinations                                                 |

Medical events carry their own color, set when the event is created.
